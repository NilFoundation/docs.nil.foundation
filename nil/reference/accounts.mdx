# Accounts JSON-RPC reference

## GetBalance

Returns the balance of the account with the given address.

### Inputs

| Parameter | Type | Description |
|:--:|:--:|:--|
| `shardId` | `ShardId` | The ID of the shard where the address is located. |
| `address` | `Address` | The address of the account whose balance is requested. |
| `blockNrOrHash` | `BlockNumberOrHash` | The number/hash of the block at which the account balance is requested.  |
