| [MetaMaskTransport](classes/MetaMaskTransport.mdx) | MetaMask transport represents the MetaMask transport for connecting to the network. MetaMask transport can be used in browser only. MetaMaskTransport |
| [PublicClient](classes/PublicClient.mdx) | PublicClient is a class that allows for interacting with the network via the JSON-RPC API. It provides an abstraction of the connection to =nil;. PublicClient enables using API requests that do not require signing data (or otherwise using one's private key). |
| [WalletV1](classes/WalletV1.mdx) | WalletV1 is a class used for performing operations on the cluster that require authentication. |
