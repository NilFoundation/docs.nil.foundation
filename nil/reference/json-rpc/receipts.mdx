# Receipts JSON-RPC reference

## `getInMessageReceipt`

Returns the receipt for the message with the given hash.

| Parameter | Type | Description |
|:--:|:--:|:--|
| `shardId` | `ShardId` | The ID of the shard where the message was included in a block. |
| `hash` | `Hash` | The hash of the message whose receipt is requested. |
